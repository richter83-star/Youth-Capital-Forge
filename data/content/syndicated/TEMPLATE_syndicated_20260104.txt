## Dashboard Overview

**Type:** Page

**Properties:**
- **Total Net Worth** (Formula): `prop('Assets Value') - prop('Liabilities Value')`
- **Daily Cash Flow** (Rollup): Sum of Net Cash Flow from Daily Metrics
- **Goal Progress** (Relation): Links to Goal Tracking database
- **Last Updated** (Last Edited Time): Auto-updates on changes

**Views:**
- **Main Dashboard** (Gallery): Full-width layout showing all key metrics

---

## Daily Metrics Tracking Sections

**Type:** Database

**Properties:**
- **Date** (Date): Entry date
- **Income** (Number): Daily income amount
- **Expense** (Number): Daily expense amount
- **Savings** (Formula): `prop('Income') - prop('Expense')`
- **Net Cash Flow** (Formula): `prop('Income') - prop('Expense') - prop('Investments')`
- **Investments** (Number): Amount invested
- **Category** (Multi-select): Salary, Business, Dividends, Rental, Other
- **Notes** (Text): Additional details

**Views:**
- **Table View**: Sorted by Date (descending)
- **Calendar View**: Visual calendar with date property

---

## Weekly & Monthly Summaries

**Type:** Linked Database (from Daily Metrics)

**Properties:**
- **Week** (Formula): `formatDate(prop('Date'), "wo")`
- **Month** (Formula): `formatDate(prop('Date'), "MMMM YYYY")`
- **Total Income** (Rollup): Sum of Income
- **Total Expense** (Rollup): Sum of Expense
- **Net Savings** (Rollup): Sum of Savings

**Views:**
- **Weekly Summary**: Grouped by Week
- **Monthly Summary**: Grouped by Month

---

## Goal Tracking

**Type:** Database

**Properties:**
- **Goal Name** (Title): Name of the financial goal
- **Target Amount** (Number): Goal target
- **Current Amount** (Rollup): Sum of Savings from Daily Metrics
- **Progress %** (Formula): `if(prop('Target Amount') == 0, 0, prop('Current Amount') / prop('Target Amount') * 100)`
- **Deadline** (Date): Target completion date
- **Status** (Select): Not Started, In Progress, Completed, Overdue
- **Related Metrics** (Relation): Links to Daily Metrics entries

**Views:**
- **Goal Tracker** (Board): Grouped by Status
- **Progress Chart** (Gallery): Shows Goal Name, Progress %, Deadline

---

## Income/Expense Categories

**Type:** Database

**Properties:**
- **Category** (Select): Salary, Business, Dividends, Rental, Utilities, Marketing, Travel, Food, Taxes, Other
- **Type** (Select): Income or Expense
- **Amount** (Number): Transaction amount
- **Date** (Date): Transaction date
- **Linked Entry** (Relation): Links to Daily Metrics

**Views:**
- **Category Summary** (Table): Grouped by Category
- **Expense Pie** (Gallery): Visual breakdown by Category and Amount

---

## Asset Tracking

**Type:** Database

**Properties:**
- **Asset Name** (Title): Name of asset
- **Category** (Select): Real Estate, Stocks, Bonds, Crypto, Business, Other
- **Current Value** (Number): Current market value
- **Acquisition Date** (Date): When asset was acquired
- **Acquisition Cost** (Number): Original purchase price
- **Growth %** (Formula): `if(prop('Acquisition Cost') == 0, 0, (prop('Current Value') - prop('Acquisition Cost')) / prop('Acquisition Cost') * 100)`
- **Notes** (Text): Additional details

**Views:**
- **Asset Overview** (Table): Sorted by Current Value (descending)
- **Growth Chart** (Gallery): Shows Asset Name and Growth %

---

## Progress Visualization

**Type:** Page

**Properties:**
- **Net Worth Over Time** (Chart - Line): Source from Daily Metrics, X-axis: Date, Y-axis: Net Cash Flow
- **Income vs Expense** (Chart - Bar): Source from Daily Metrics, X-axis: Date, Y-axis: Income and Expense
- **Goal Completion** (Chart - Progress): Source from Goal Tracking, Value: Progress %
